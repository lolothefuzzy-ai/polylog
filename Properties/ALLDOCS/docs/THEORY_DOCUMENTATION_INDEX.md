- Computational complexity: O(n log n) per evaluation

## Key Metrics to Remember

### Practical Size Limits
- **Perfect:** n < 10 polyforms, instant computation
- **Good:** n = 10-30 polyforms, seconds per generation
- **Acceptable:** n = 30-50 polyforms, minutes per generation
- **Degraded:** n = 50-100 polyforms, hours per generation
- **Impractical:** n > 100 polyforms, exponential slowdown

### Assembly Order Limits
- **Observed plateau:** Order 20-50
- **Topological limit:** Order ~100
- **Physical reality:** Way lower (material constraints)

### Quality Metrics
- **Solution quality:** 60-80% of theoretical optimum
- **Convergence reliability:** 90%+ for >100 generations
- **Reproducibility:** 100% (deterministic with seed)
- **Scalability:** O(10^n) exponential
